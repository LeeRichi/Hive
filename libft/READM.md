### Instructions

**1. Compiling the library**

To compile, go to the library path and run:

```shell
$ make
```

Other make-files-commands:
```shell
$ make bonus
$ make clean
$ make fclean
$ make re
```
**2. Using it in your code**

To use the library functions in your code, simply include its header:

```C
#include "libft.h"
```
